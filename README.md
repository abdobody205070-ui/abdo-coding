# 🚀 Abdo Coding Platform

منصة تعليم برمجة احترافية مبنية بـ HTML + CSS + JavaScript مع نظام تسجيل دخول حقيقي باستخدام Firebase.

---

## 📌 نبذة عن المشروع

**Abdo Coding** هي منصة تعليمية بسيطة ولكن احترافية تهدف لتعليم البرمجة من الصفر حتى الاحتراف من خلال:

- كورسات منظمة
- فيديوهات من YouTube
- نظام تسجيل دخول حقيقي
- تتبع تقدم الطالب
- تصميم حديث قريب من Coursera

---

## 🎯 المميزات

### 🔐 نظام المستخدمين
- تسجيل حساب جديد
- تسجيل دخول آمن باستخدام Firebase
- حفظ الجلسة تلقائيًا

### 📚 نظام الكورسات
- عرض الكورسات بشكل شبكي (Grid)
- كورسات HTML / CSS / JavaScript
- إمكانية فتح كل كورس

### 🎥 تشغيل الفيديوهات
- دمج فيديوهات YouTube مباشرة داخل المنصة

### 📊 تتبع التقدم
- حفظ نسبة التقدم باستخدام Local Storage
- عرض Progress للمستخدم

### 🎨 التصميم
- تصميم Dark Modern احترافي
- واجهة مشابهة لمنصات مثل Coursera

---

## 🛠️ التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Firebase Authentication
- YouTube Embed
